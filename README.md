# 📊 Dashboard de Compras - Power BI

## 📌 Sobre o Projeto
Este projeto consiste no desenvolvimento de um dashboard interativo no Power BI para análise de compras, fornecedores e produtos.

O objetivo é transformar dados em insights estratégicos para apoiar a tomada de decisão.

---

## 🎯 Objetivos
- Analisar o volume e valor de compras
- Identificar principais fornecedores
- Avaliar produtos com maior impacto financeiro
- Monitorar variações ao longo do tempo

---

## 📊 Principais Análises
- Compras por mês
- Ranking de fornecedores
- Compras por produto
- Indicadores gerais (KPIs)


## 🛠️ Ferramentas Utilizadas
- Power BI
- DAX
- Modelagem de Dados

---

## 📈 Insights Obtidos
- Alta concentração de compras em poucos fornecedores
- Variação significativa ao longo dos meses
- Produtos com maior impacto financeiro claramente identificados

---

## 🚀 Como usar
1. Baixe o arquivo `.pbix`
2. Abra no Power BI Desktop
3. Explore os filtros e análises
